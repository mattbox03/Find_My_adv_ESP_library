# Security policy

Tracker advertisement keys are sensitive identifiers. Never include real keys
in public issues, pull requests, screenshots, logs, example sketches, or test
binaries.

For a suspected vulnerability, contact the repository owner privately through
the security-reporting mechanism shown on GitHub. Include the affected library
version, ESP target, and a minimal reproduction using synthetic keys.

FindMyAdv does not persist or transmit configuration over IP. Security of web
forms, MQTT, Preferences/NVS, filesystems, and remote update mechanisms remains
the responsibility of the host firmware.
